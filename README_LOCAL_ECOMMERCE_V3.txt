本機電商整合測試版 V3

本版將「加入電商資料」改為全報表口徑切換：

1. 未啟用時：Dashboard、每日總覽、商品日銷售趨勢、單一商品趨勢分析皆只看現場資料。
2. 啟用時：上述四個區塊皆加入電商資料。
3. 電商報表目前只有數量，金額用現場銷售單價回推：
   - 優先使用同商品同日期的現場含稅金額 / 現場數量
   - 若同日期現場沒有銷售，使用該商品整段期間平均現場單價
4. 單一商品趨勢分析中的「平均每日現場銷售數量 / 金額」與「中位數現場銷售數量 / 金額」仍只計算現場資料，不會因加入電商資料而改變。
5. 每日總覽在啟用電商後會顯示現場、電商、合併三組欄位。

本機測試建議：
cd hunter_kaohsiung-main
python -m http.server 8000
瀏覽器開啟 http://localhost:8000
