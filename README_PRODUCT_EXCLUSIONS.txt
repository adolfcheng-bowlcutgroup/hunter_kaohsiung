產品代號排除清單使用方式
========================

若有品項不屬於商售範圍，但仍會出現在每日產銷報表或電商報表中，請不要刪除原始 CSV。
請改為維護：

data/excluded_products.json

格式如下：

{
  "productCodes": [
    "請填入產品代號1",
    "請填入產品代號2",
    "請填入產品代號3"
  ]
}

範例：

{
  "productCodes": [
    "HUKHA00526",
    "HUKHXXXX01",
    "HUKHXXXX02"
  ]
}

注意事項：
1. 只要產品代號列在 productCodes 中，該商品會從現場銷售資料與電商資料中排除。
2. 被排除的商品不會進入 Dashboard、每日總覽、商品日銷售趨勢、單一商品趨勢分析與匯出 CSV。
3. 原始 CSV 可以照常保留與更新，不需要人工刪列。
4. 修改 excluded_products.json 後，在頁面按「重新讀取 data 資料夾」即可套用。
5. 產品代號比對會自動忽略前後空白，大小寫不敏感。
